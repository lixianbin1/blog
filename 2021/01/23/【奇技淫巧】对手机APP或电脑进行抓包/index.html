对手机APP或电脑进行抓包

下载 Fiddler 软件

可以从官网下载，或者下载本地汉化版本

运行 Fiddler ,点击 tools → options → https ,勾选如图

使用教程

手机连上电脑的WIFI(既在同一局域网中)，设置wifi代理 IP为电脑IP 端口是Fiddler端口
如果都是http请求，能直接抓到，如果是https请求，手机就需要下载证书。打开手机浏览器输入：http://10.224.xx.xx:8888 点击下载
点击左下角的图标，空白则只抓取远程手机端的包，如果不点则默认手机PC端都会抓取

或者设置过滤：打开fiddler>Tools>Options>HTTPS>勾选Decrypt Https traffic>选择…from remote clients only,勾选这个选项就可以了
…from all processes :抓所有的请求
…from browsers only ：只抓浏览器的请求
…from non-browsers only :只抓非浏览器的请求
…from remote clients only:只抓远程客户端请求
