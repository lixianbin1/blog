软件的端口占用等端口问题

查看使用中的端口

状态
LISTENING 处于侦听状态，端口是开放的，等待连接，但还没有被连接
ESTABLISHED 建立连接。表示两台机器正在通信
TIME_WAIT 结束了这次连接。说明端口曾经有过访问，但访问结束了
SYN_SENT 表示请求连接，SYN_SENT状态非常短暂
CLOSE_WAIT 对方主动关闭连接或者网络异常导致连接中断，我方状态
LAST-ACK 等待原来的发向远程TCP的连接中断请求的确认
…
端口 意义
0.0.0.0 所有未知ip的地址的集合
127.0.0.1 回送地址，本地机
192.168.0.1 属于C类地址中的一个私有地址
192.168.1.1 属于C类地址中的一个私有地址
[::] [::]就是全0的ip，相当于是ipv4中的0.0.0.0

执行命令窗口卡住不刷新界面

执行命令窗口卡住，回车后一顿输出,不回车就卡在这个界面，原因是当前命令窗口可能开启了快速编辑选项，当你点击窗口就进入了编辑模式，后台执行就会被卡住，

1.右键窗口，选择属性，将编辑选项中的快速编辑功能关闭即可

sublime 无法执行安装软件包

在 Ctrl + Shift + P 或者点击 首选项 - Package Control 后，使用install Package 进行安装插件命令，有时候发现连不上 https://packagecontrol.io/channel_v3.json 这是网络问题

点击 首选项 - Package Settings - Package Control - Settings user
