Hexo及markdown的语法参考

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页，而Markdown是一个面向网络编写者的文本到HTML转换工具。Markdown允许您使用易于阅读，易于编写的纯文本格式进行编写，然后将其转换为结构有效的XHTML（或HTML）

Hexo

Front-matter

Front-matter 是文件最上方以 — 分隔的区域，用于指定个别文件的变量，举例来说

---
layout: post       # 布局
title: Hello World # 标题
date: 2013/7/13 20:46:25    # 建立日期
updated: 2013/7/13 20:46:27 # 更新日期
comments: true     # 开启文章的评论功能
tags: css          # 标签（不适用于分页）
categories: css    # 分类（不适用于分页）
permalink: wwww.xxx.com     # 覆盖文章网址
---

内网应用

有时候你需要引用自己写的文章，那可能只能使用 Hexo 的制定内容

{% post_link 文章名称去掉md的 显示名称可选 %}
{% jsfiddle xianbin/bu5ntf8m/1/ result,html,css %}

外网或者示例

  • jsFiddle 代码在线演示的网站，将示例代码展示在上面，由于国内网络问题，可能显示困难
{% jsfiddle xianbin/bu5ntf8m/1/ result,html,css %}
  • Gist GitHub代码片段演示网站，在文章中嵌入 Gist。由于国内网络问题，可能显示困难
<script src="https://gist.github.com/lixianbin1/aeb46f5571f6291fdf05ec5a753bdada.js"></script>
  • iframe 兼容性查询网站，在文章中插入 iframe
{% iframe https://caniuse.com/css-variables/embed 100% 495px %}

图片及链接

# 图片
{% img [class names] /images/posts/1911051.jpg [width] [height] [title text [alt text]] %}

# link
# 在文章中插入链接，并自动给外部链接添加 target="_blank" 属性。
{% link text url [external] [title] %}

video

# Youtube 一个视频媒体播放网站
# 在文章中插入 Youtube 视频。
{% youtube video_id %}

# Vimeo 一个高清视频播客网站
# 在文章中插入 Vimeo 视频。
{% vimeo video_id %}
