学习及乱七八糟的网站

博客论坛平台

  • 阮一峰 http://www.ruanyifeng.com/blog/
    他写的文章都比较好懂，能用简单的话语把一件复杂的事情讲清楚，不像是网上有的文章说的是中国话，我却不能理解。

  • 掘金 https://juejin.im/
    对于开发者的一个开拓视野，学习知识的一个平台，但你不能指望从这里去学习你想要的知识

  • V2EX https://www.v2ex.com/?r=VWMMWV
    一个开发者的问与答的社区，你可以在里面提问或帮助某人解决问题。氛围还不错，但你还是要抱有坚持

  • 淘宝FED http://taobaofed.org/
    淘宝前端团队(FED) ,你可以在里面了解他们的开发技术及工具，了解前沿的工作团队。

  • codrops https://tympanus.net/codrops/
    一个网页设计和开发的博客,发布关于最新的网络趋势,技术和新可能性的文章和教程

  • StackOverflow https://stackoverflow.com/
    Stack Overflow是一个开放社区，适合任何编码的人。它可以帮助你获得最棘手的编码问题的答案

  • 谷歌实验室 https://experiments.withgoogle.com/
    一个用来演示Google还在测试阶段的新项目产品的网站。里面的东西稀奇古怪，技术也很新颖

  • coderbyte https://www.coderbyte.com/courses/
    一个考验解答问题和培养逻辑的测试学习网站，里面的题目由简到难，触及各个方面

文档手册培训

  • MDN web docs https://developer.mozilla.org/zh-CN/docs/Web
    一个完整的Web学习平台，你可以在这里深入学习Web技术以及能够驱动Web的软件,它很完整，冷门的知识点也概括

  • W3C中国 http://www.chinaw3c.org/
    万维网联盟（World Wide Web Consortium，简称W3C）旨在通过促进通用协议的发展并确保其通用型

  • W3Cschool https://www.w3cschool.cn/ HTML中文网 https://www.html.cn/
    以视频教程和文档手册为主的网站，你可以用来学习一些比较基础的知识点，

  • 菜鸟教程 https://www.runoob.com/
    一个文档手册为主的学习平台，手册可能不是很深入但很适合入门，如果你刚开始学习，为推荐它

  • ES6 https://es6.ruanyifeng.com/
    一本由阮一峰所著的开源的ES6技术书，希望有条件可以选择支持下实体书籍的销量

  • babel https://github.com/jamiebuilds/babel-handbook/blob/master/translations/zh-Hans/user-handbook.md
    这本手册涵盖了关于 Babel 的使用及其相关工具的内容。（Babel是用于解决ES6语法兼容性的工具）

各类框架文档

  • Angular https://www.angular.cn/start
    Angular框架的文档及介绍

  • Vue https://cn.vuejs.org/v2/guide/
    Vue框架的官方文档及介绍

  • React https://react.docschina.org/docs/getting-started.html

(PS:这里就列个样子，不然的话，列出来的东西太多来)

不好分类的网站

  • 作业部落 https://www.zybuluo.com/
    平时用来记下东西，写下笔记啥的，反正我一开浏览器，首先就打开的网站（习惯了）
