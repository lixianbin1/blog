电脑装系统的过程及遇到的问题

安装系统

当你拿着一个新的固态硬盘，然后手头上又只有一台连着网络的电脑的时候，一堆人让你拿U盘做PE启动盘，那要是没有呐。

1：去官网上下一个ios镜像系统。或者你找一个系统之家下载一个你认为安全的系统ios镜像
2：下载一个WinNTSetup,然后打开，选择ios文件，选择你要安装系统的系统引导的ESP区(你放在哪个盘，哪个盘开机就是必须的)，再选择你需要安装在哪个盘。
3：安装完成后，开机进入你安装的系统，看你需要双系统还是不需要，如果需要就不用再动了，如果不需要就打开系统设置(左下角搜索msconfig)。进入引导，把不需要的系统删除，然后重启。

激活系统

以下是Win10专业版系统的激活方式:

鼠标右击开始，选择Windows PowerShell（管理员）,点击打开
输入slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX 按回车键
输入slmgr /skms kms.03k.org 按回车键
输入slmgr /ato按回车键

遇到的问题

双系统，把旧固态卸载了，就启动不了

你的新固态的引导区在旧固态上，新固态安装的系统，但是把旧固态下了，没有系统引导区就启动不了。

解决方案：重新装回新系统，使用 DiskGenius, 在你的新固态上建立一个 100M 或者 300M 的 ESP区，克隆你旧固态上的 ESP区 到新固态的 ESP区。然后把旧固态下了，进入新固态的系统后打开系统设置(左下角搜索msconfig)。进入引导，把不需要的系统删除，然后重启。

Boot 上没看到我的固态，设置不了启动项

开机的时候，选择 F2 进入 Bios ,切换到 Advanced 选择 Os Support 菜单，切换到 Legacy OS, 然后 F4 重启。再次进入 F2 ,切换到 Boot ,滚动到底部，选择 Haed Disk Drive BBS Prior ities 进入选择，选择你的固态到第一个启动项，然后回车回到 Boot 界面，就可以在 Boot 中看到你的固态了。
** 注意：直接使用WinNtSetup安装的系统好像只能通过UEFI启动,不能设置 Legacy OS**

Win10系统桌面怎么没有我的电脑

右键进入个性化设置，选择主题，点击桌面图标设置，然后勾选你想要在桌面显示的功能，最后应用即可。
