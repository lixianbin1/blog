VueUse基于组合式的实用工具

安装

yarn add @vueuse/core || npm i @vueuse/core

核心函数

State

createGlobalState 全局状态

将状态保存全局作用域中，以便跨Vue实例复用。

createGlobalState的使用

createInjectionState 注入全局状态

创建可以注入到组件中的全局状态。

useStorage 持久存储

响应式 LocalStorage/SessionStorage

useLocalStorage 持久存储

响应式的 LocalStorage

用法同 useStorage

useSessionStorage 持久存储

响应式SessionStorage

用法同 useStorage

useStorageAsync 异步存储

支持异步的响应式Storage

用法同 useStorage

createSharedComposable

让一个钩子函数可用于多个Vue实例中。

useAsyncState 异步状态

响应式获取异步状态。不会阻塞setup 函数，在promise完成后，将自动触发。

useLastChanged 最后时间

记录最后一次更改的时间戳

useRefHistory 历史记录

跟踪 ref 的更改历史，提供撤消和重做功能

useManualRefHistory 手动历史记录

调用 commit() 时，手动跟踪ref的更改历史，提供撤消和重做功能

useThrottledRefHistory 节流历史记录

带节流过滤器的 useRefHistory 的简写。

useDebouncedRefHistory 防抖历史记录

useRefHistory 的简写，带有防抖过滤器。

Element

useActiveElement

响应式 document.activeElement

useDocumentVisibility

响应式跟踪 document.visibilityState

useDraggable 元素拖拽

使元素可拖拽

useDropZone

创建一个可以放置文件的区域。

useElementBounding

让HTML元素的bounding box 响应式

useElementSize

元素尺寸大小响应式

useElementVisibility

跟踪元素在视口中的可见性。

useIntersectionObserver

响应式监听目标元素的可见性。

useMouseInElement

响应式获取鼠标相对于元素的位置

useMutationObserver

监听DOM树修改 MutationObserver MDN

useResizeObserver

监听元素内容和边框尺寸的变化。

useWindowFocus

使用 window.onfocus 和 window.onblur 事件响应式跟踪窗口焦点。

useWindowScroll

响应式获取窗口的滚动位置。

useWindowSize

响应式获取窗口尺寸

组件中的使用

Store

defineStore

Store 是用 defineStore() 定义的，它的第一个参数要求是一个独一无二的名字

可以使用Setup语法，也可以使用配置语法，推荐配置语法，setup会让SSR变得更复杂。

state

  • 使用示例

getter

Getter 完全等同于 store 的 state 的计算值。可以通过 defineStore() 中的 getters 属性来定义它们

Action

Action 相当于组件中的 method。它们可以通过 defineStore() 中的 actions 属性来定义

插件

插件是通过 pinia.use() 添加到 pinia 实例的
