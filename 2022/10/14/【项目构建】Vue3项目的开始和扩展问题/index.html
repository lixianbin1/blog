Vue3 项目的开始和扩展问题

Vue 3 是 Vue 当前的最新主版本。它包含了一些 Vue 2 中没有的新特性 (比如 Teleport、Suspense，以及多根元素模板)。同时它也包含了一些与 Vue 2 非兼容性的变更

使用须知

文中举例，默认 yarn 和 vite 是已经安装了，如果没有安装，请先运行以下命令安装后；

npm install yarn -g
yarn create vite || npm init vue@latest

vite在创建项目的时候会提示配置项：既第三方库

开启项目

vite 在创建项目xxx后,可以命令进入，vite默认启动的IP是错误的，可以通过配置进行修改；

cd xxx
yarn
yarn dev

Vite

Vite是一个前端构建工具，它包含一个开发服务器，一套构建指令。vite 以当前工作目录作为根目录启动开发服务器。你也可以通过 vite serve some/sub/dir 来指定一个替代的根目录。 注意的是 index.html 与项目根目录，vite以index.html作为入口文件。也支持多个 .html 作入口点的 多页面应用模式

TypeScript

vite天然支持TypeScript，Vite 仅执行 .ts 文件的转译工作，并 不 执行任何类型检查。并假设类型检查已经被你的 IDE 或构建过程接管了。
