Vue.js 3的学习笔记

Vue 3 是 Vue 当前的最新主版本。它包含了一些 Vue 2 中没有的新特性 (比如 Teleport、Suspense，以及多根元素模板)。同时它也包含了一些与 Vue 2 非兼容性的变更

简单使用

可以使用vue2版本的选项式编写，也可用使用vue3新添加setup函数

index.vue
<script>
import { ref } from 'vue'

export default {
  setup() { //setup函数
    const count = ref(0) //类似data 数据
    return { // 返回值会暴露给模板和其他的选项式 API 钩子
      count
    }
  },
  mounted() { //生命周期钩子
    console.log(this.count) // 0
  }
}
</script>

<template>
  <button @click="count++">{{ count }}</button>
</template>

setup() 函数

index.vue
<script>
export default {
  setup() { },
  //...
}
</script>

或者

index.vue
<script setup lang="ts">

</script>

差异点

响应式数据

ref和reactive是vue3基于组合式api模式下，在setup中用于声明的具有响应式的数据的方法, ref通常用于声明基础类型响应式数据,reactive用于声明复杂类型响应式数据

index.vue
<script setup>
  import { reactive } from 'vue'
  const state = reactive({ count: 0 })
  state.count++ //数据的改变
  const name = ref(21)
  name.value=12
</script>
<template>
  <div>{{ state.count }}</div>
</template>

选项式代码则和Vue2相同

index.vue
<script>
export default {
  data() {
    return {
      count: 1
    }
  },
  mounted() {
    this.count++
  }
}
</script>
<template>
  <div>{{ count }}</div>
</template>

计算属性

index.vue
<script setup>
import { reactive, computed } from 'vue'
const state = reactive({
  name: 'John Doe',
})
const publish = computed(() => { //计算属性
  return state.name > 0 ? 'Yes' : 'No'
})
</script>
<template>
  <div>{{ publish }}</div>
</template>

选项式代码则和Vue2相同

export default {
  data() {
    //...
  },
  computed: {
    //...   
  }
}

侦听器

组合式代码

index.vue
import { ref, watch } from 'vue'

const question = ref('')

watch(question, async (new, old) => {
  console.log(new, old)
})

选项式代码

index.vue
watch: {
  question(new, old) {
    console.log(new, old)
  }
}

ref引用

index.vue
import { ref, onMounted } from 'vue'
// 声明一个 ref 来存放该元素的引用
// 必须和模板里的 ref 同名
const input = ref(null)
onMounted(() => {
  input.value.focus()
})

// ...
<input ref="input" />

$emit

const emit = defineEmits({
  // 没有校验
  click: null,

  // 校验 submit 事件
  submit: ({ email, password }) => {
    if (email && password) {
      return true
    } else {
      console.warn('Invalid submit event payload!')
      return false
    }
  }
})

function submitForm(email, password) {
  emit('submit', { email, password })
}

自定义指令

指令比 autofocus attribute 更有用，因为它不仅仅可以在页面加载完成后生效，还可以在 Vue 动态插入元素后生效

index.vue
// 在模板中启用 v-focus
const vFocus = {
  mounted: (el) => el.focus()
}
//...
<template>
  <input v-focus />
</template>

问题

ref 和 reactive 的区别

reactive：(1)它的响应式是更加‘深层次’的，底层本质是将传入的数据包装成一个Proxy。(2)参数必须是对象或者数组，如果要让对象的某个元素实现响应式时比较麻烦。需要使用toRefs (3)更适合复杂对象

ref：(1)函数参数可以是基本数据类型，也可以接受对象类型。(2)如果参数是对象类型时，其实底层的本质还是reactive,系统会自动根据我们给ref传入的值转换成(3)更适合简单类型
